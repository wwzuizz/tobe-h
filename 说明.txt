npm create vue@latest

#出现这个错误：npm : 无法加载文件 D:\nodejs\npm.ps1，因为在此系统上禁止运行脚本
执行Set-ExecutionPolicy Bypass -Scope Process

要先登录（因为翻墙原因，所以最好使用翻墙软件的命令行）
firebase login

要先把代码放到github

1、安装这个后可以全局使用firebase，也可以直接下载firebase，然后命令行执行。（因为翻墙原因，所以最好使用翻墙软件的命令行）
npm install -g firebase-tools

2、项目根目录执行
firebase init hosting

Are you ready to proceed? Yes
是否指定目录

是否通过github
Set up automatic builds and deploys with GitHub? Yes
选择仓库
For which GitHub repository would you like to set up a GitHub workflow? (format: user/repository) wwzuizz/tobe-h
是否自动部署，需要npm等支持

是否所以分支改变就更新
Set up automatic deployment to your site's live channel when a PR is merged? Yes

What is the name of the GitHub branch associated with your site's live channel? main
